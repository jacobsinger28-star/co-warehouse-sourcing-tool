The Off-Market Operating System
NextAutomation, nextautomation.us

A guide and a drop-in Claude kit for sourcing off-market real estate deals in any
asset class. Paste it into a Claude Project and start the same day, no code.

Start with 00-start-here.md. Then paste 01-claude-project-setup.md into your
Claude Project's custom instructions, and fill 02-deal-box-template.md and add it
as a Project file.

  01-claude-project-setup.md  The paste-in system prompt (the drop-in core)

  03-the-four-stage-engine.md Find -> Score -> Resolve -> Reach, with the prompts
  04-signal-library.md        Cross-asset signals and point weights
  05-county-data-and-the-law.md  Data tiers, free portals vs providers, the law

  07-owner-resolution.md      LLC-to-human, evidence chains, the honest ceiling

  09-from-list-to-deals.md    Cadence, re-running, and the maintenance truth
